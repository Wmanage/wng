A C package manager, project manager and build tool.

+ [ ] Project creation
+ [ ] Project type (~binary~, ~shared~ or ~static~)
+ [ ] Standard selection
+ [ ] Compiler selection
+ [ ] Release mode
+ [ ] Custom build script
+ [ ] Dependencies management
  - [ ] Branch selection
  - [ ] Commit selection
+ [ ] Package installation
